**Thumb Combo Reference:**
- **LA + LO** = ALT
- **LO + RE** = CTRL  
- **LA + RU** = ESC
- **RE + RU** = BSPC
- **RU + RNO** = TAB
- **LA + LO + RE + RU** (all 4) = CAPS LOCK

## PWR Layer - Hold PWR (Bottom Left) + Any Key

```
     ┌─────┬─────┬─────┬─────┬─────┬─────┐     ┌─────┬─────┬─────┬─────┬─────┬─────┐
     │     │  !  │  @  │  {  │  }  │  |  │     │  +  │  &  │  *  │  .  │  ,  │ TAB │

     │ PWR │  %  │  ^  │  [  │  ]  │  ~  │     │  '  │  =  │  -  │  /  │  \  │     │
     │HOLD!│     │     │     │     │     │     │     │     │     │     │     │     │
     └─────┴─────┴─────┴─────┴─────┴─────┘     └─────┴─────┴─────┴─────┴─────┴─────┘

                       └─────┴─────┴─────┘     └─────┴─────┴─────┘
```

**PWR Symbol Reference:**
- Top row: **! @ { } | + & * . , TAB**
- Middle row: **# $ ( ) ` "**
- Bottom row: **% ^ [ ] ~ ' = - / \**
- Thumbs: **SHIFT / ; \**

**Right Hand Modifiers:**
- **RT + RD** (P + ;) = CTRL
- **RS + RZ** (/ + ESC) = ALT
- **RT + RD + RS + RZ** (all 4) = Windows/Command Key (LGUI)

## Quick Reference Summary

```
╔══════════════════════════════════════════════════════════════════╗
║                   STENOMI QUICK REFERENCE                        ║
╠══════════════════════════════════════════════════════════════════╣
║                                                                  ║
║  VERTICAL COMBOS → Letters (A-L)                                ║
║     Press top + bottom in same column (Q+Z=A, W+X=S, etc.)      ║
║                                                                  ║
║  THUMB COMBOS → Modifiers                                       ║
║     LA+LO=ALT | LO+RE=CTRL | LA+RU=ESC | RE+RU=BSPC | RU+RNO=TAB║
║                                                                  ║
║  HOLD LNO (left thumb) → Numbers & Functions                    ║
║     + Top row = 1-0  |  As FUNC = F1-F12                        ║
║                                                                  ║
║  HOLD PWR (bottom left) → Symbols                               ║
║     + Any key = symbol (see PWR layer chart above)              ║
║     + Two keys = arrows (U+M=←, I+,=↓, O+.=↑, P+;=→)           ║
║                                                                  ║
║  4-KEY COMBOS → Special Layers                                  ║
║     Z+X+C+V = Media  |  T+B = Movement                          ║
║     RT+RD+RS+RZ = Windows/Command key                           ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```
